# Zadanie 2: Interakcja w funkcji

## Cel

Nauczysz się, jak sprawić, by funkcja pobierała dane (`prompt`) i wyświetlała je w HTML.

## Instrukcja

1.  W pliku `index.html` w sekcji `<head>` zdefiniuj funkcję `przedstawSie`.
2.  Tworząc kod **wewnątrz** funkcji:
    - Użyj `prompt()`, aby zapytać użytkownika o imię.
    - Użyj `document.write("Miło cię widzieć, " + imie + "<br>");`, aby wypisać powitanie.
3.  Zjedź do sekcji `<body>`. Znajdź element `<div id="wynik">`.
4.  Wewnątrz tego diva wstaw znacznik `<script>` i wywołaj w nim swoją funkcję: `przedstawSie();`. Możesz wywołać ją dwa razy, aby zobaczyć efekt.
5.  **W stopce pliku HTML (w sekcji `<footer>`) wpisz swoje imię, nazwisko i klasę.**
