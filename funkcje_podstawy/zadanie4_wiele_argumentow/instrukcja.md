# Zadanie 4: Wiele argumentów

## Cel

Nauczysz się tworzyć funkcje przyjmujące więcej niż jeden argument.

## Instrukcja

1.  W pliku `index.html` w sekcji `<head>` zdefiniuj funkcję `poleProstokata(a, b)`.
2.  Wewnątrz funkcji oblicz pole: `let wynik = a * b;`.
3.  Użyj `document.write("Pole prostokąta o bokach " + a + " i " + b + " wynosi " + wynik + "<br>");`.
4.  Zjedź do sekcji `<body>`. Znajdź element `<div id="wynik">`.
5.  Wewnątrz tego diva wstaw znacznik `<script>` i wywołaj funkcję dla różnych par liczb (np. `5, 10` oraz `2.5, 4`).
6.  **W stopce pliku HTML (w sekcji `<footer>`) wpisz swoje imię, nazwisko i klasę.**
7.  Pamiętaj o kolejności argumentów - pierwszy argument trafi do `a`, drugi do `b`.
