# Zadanie 3: Parametry funkcji

## Cel

Nauczysz się przekazywać dane do funkcji przy użyciu **parametrów**.

## Instrukcja

1.  W pliku `index.html` w sekcji `<head>` zdefiniuj funkcję `powitajUcznia(imie)`.
2.  Wewnątrz funkcji użyj `document.write("Uczeń: " + imie + "<br>");`, aby wypisać powitanie.
3.  Zjedź do sekcji `<body>`. Znajdź element `<div id="listaUczniow">`.
4.  Wewnątrz tego diva wstaw znacznik `<script>` i wywołaj w nim funkcję trzy razy dla różnych imion.
5.  **W stopce pliku HTML (w sekcji `<footer>`) wpisz swoje imię, nazwisko i klasę.**
